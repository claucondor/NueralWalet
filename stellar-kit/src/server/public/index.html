API Documentation

This interface uses the following endpoints:

  • /api/account/generate - Generate a new keypair
  • /api/account/:publicKey - Get account information
  • /api/account/fund - Fund an account with testnet XLM
  • /api/transaction/payment - Send a transaction

For more details about available endpoints, check the server code.
