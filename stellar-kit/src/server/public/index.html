Esta interfaz usa los siguientes endpoints:

  • /api/account/generate - Genera un nuevo par de claves
  • /api/account/:publicKey - Obtiene información de una cuenta
  • /api/account/fund - Fondea una cuenta con XLM de testnet
  • /api/transaction/payment - Envía una transacción

Para más detalles sobre los endpoints disponibles, consulta el código del servidor.
